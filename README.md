# Personal Information Manager

## Project Description
This is a beginner-level Python program created as part of Week 1 learning.
The program stores and displays personal information using Python basics.

## What I Learned
- Variables and data types
- Taking user input
- Printing output
- String formatting using f-strings
- Basic input validation
- Simple calculations

## How to Run the Program
1. Install Python from https://www.python.org
2. Open terminal or command prompt
3. Navigate to the project folder
4. Run the command:
   python personal_info.py
5. Enter details when asked

## Features
- Stores static personal details
- Takes user preferences as input
- Validates empty input
- Calculates age in months
- Displays formatted output
